Вариант 1
интегрировать React и загружать тесты с сервера. Вопросы тестов загружать и обрабатывать

    по одному
        После выбора теста загружается описание и кнопка ПРОЙТИ ТЕСТ. При нажатии на кнопку "ПРОЙТИ ТЕСТ" загружается
            все вопросы теста сразу
            вопросы по одному


    все вопросы теста сразу
        по запросу из листа тестов по API загружается JSON тест целиком. На фронте ответы на вопросы происходят
            по отдельности
            все вопросы сразу


Варивнт 2
Реализовать с помощью django templates без js

Требование
Варианты обработки и взаимодействия с вопросами должны быть разными
    Вопрос с одним ответом
    Вопрос со множеством ответов
    Ответ текстом

Как это сделать?
Модель вопроса имеет поле выбора варианта. При создании теста можно выбрать вариант. Для разных типов вопросов
во вью использовать разные формы.
Почему? Какие варианты?

Вью
При отрисовке вопроса вариант поля вопроса сравнивается по идентификатору типа. В зависимости от варианта отрисовывается
одна из форм

Клик на вариант ответа является ответом на вопрос
